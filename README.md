# API de Cadastro de Tarefas

## Descrição
API REST para gerenciamento de tarefas com autenticação e autorização de usuários.

## Tecnologias
- ASP.NET Core
- Entity Framework Core
- ASP.NET Identity
- JWT
- MySQL
- Swagger

## Funcionalidades
- Cadastro e login de usuários
- Autenticação com JWT e refresh token
- CRUD de tarefas
- Filtros e regras de negócio
- Endpoints protegidos com autorização
